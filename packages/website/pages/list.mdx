## List

```jsx
import { useVirtual, List } from "@af-utils/react-virtual-list";

const Item = ({ i }) => <div>{i}</div>;

const SimpleList = () => {
    const model = useVirtual({
        itemCount: 50000
    });

    return <List model={model}>{Item}</List>;
};
```

## Installation

```bash
# npm
npm i -S @af-utils/styled @af-utils/react-virtual-list @af-utils/react-virtual-headless prop-types

# yarn
yarn add @af-utils/styled @af-utils/react-virtual-list @af-utils/react-virtual-headless prop-types
```

### Options

-   `model: Model`
    -   **Required**
    -   return value of <Link href="/headless#use-virtual"><a>`useVirtual()`</a></Link>
-   `children: ReactElement`
    -   **Required**
    -   React component, that will have `i` prop when rendered.
-   `itemData: Any`
    -   Will be passed as `data` prop to each `Item`
-   `getKey: Function(i, itemData) => Integer`
    -   Defaults to: `i => i`
    -   if you have unique id to pass as key - it may be more efficient.
-   All other props are passed to outermost div
