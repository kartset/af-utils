## Installation

```bash
# npm
npm i -S @af-utils/styled @af-utils/react-virtual-list @af-utils/react-virtual-headless @babel/runtime prop-types

# yarn
yarn add @af-utils/styled @af-utils/react-virtual-list @af-utils/react-virtual-headless @babel/runtime prop-types
```

## List

```jsx
<List model={model} horizontal={false}>
    {Item}
</List>
```

### Options

- `model: Model`
  - **Required**
  - return value of `useVirtual()`
- `children: ReactElement`
  - **Required**
  - React component, that will have `i` prop when rendered.
- `itemData: Any`
  - Will be passed as `data` prop to each `Item`
- All other props are passed to outermost div


export default ({ children }) => <MdxWrapper>{children}</MdxWrapper>;
